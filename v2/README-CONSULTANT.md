## 🎯 Utilisation Client par Client

### Phase 1 : Découverte (2h)

| Livrable | Action | Output |
|:---------|:-------|:-------|
| Grille Qualifier | Animer atelier express | 5-10 usages identifiés |
| Arbre Décision | Trier par niveau | Priorisation |
| Cas Limites | Identifier ambiguïtés | Plan de clarification |

### Phase 2 : Pilotage (2 jours)

| Jour | Activité | Livrable Client |
|:-----|:---------|:----------------|
| J1 | Qualifier 3 usages pilotes | 3 fiches L1/L2/L3 |
| J2 | Former référents + créer registre | Registre peuplé, équipe autonome |

### Phase 3 : Industrialisation (sur 90 jours)

| Semaine | Action | Support Consultant |
|:--------|:-------|:-------------------|
| S1-2 | Extension autonome | Support à distance |
| S3-4 | Revue + ajustements | Atelier REX |
| S5-8 | Gouvernance + automatisation | Audit interne |
| S9-12 | Institutionnalisation | Préparation certification |

## 💡 Conseils de Facilitation

### Premier Contact Client

1. **Ne pas vendre la méthode** — vendre la valeur
2. **Commencer par un usage 🟢** — victoire rapide
3. **Montrer un cas réel** — crédibilité
4. **Laisser l'équipe qualifier** — appropriation

### Gestion des Résistances

| Objection | Réponse Consultant |
|:----------|:-------------------|
| "Trop complexe" | "15 min pour le premier, on fait ensemble" |
| "Pas le temps" | "Un usage 🟢 maintenant, le reste plus tard" |
| "On le fait déjà" | "Parfait, documentons-le proprement" |
| "L'IA change" | "D'où les revues planifiées, pas l'urgence" |
