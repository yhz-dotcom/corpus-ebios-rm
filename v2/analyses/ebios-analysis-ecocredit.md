## 2. INCIDENTS CONFIRMÉS

### 2.1 Incident 1 — Paradoxe Rénovation Écologique

| Élément | Détail |
|:---|:---|
| **Situation** | Famille nombreuse, zone rurale |
| **Contraintes** | Voiture obligatoire, chauffage fioul (pas d'alternative) |
| **Score carbone** | Élevé (mauvaise empreinte) |
| **Demande** | Prêt pour rénovation isolation (devenir écologique) |
| **Décision** | **Refus** car score carbone trop élevé |
| **Paradoxe** | Refus crédit permettant de devenir écologique car pas assez écologique |

#### Analyse — Paradoxe ÉSG

```
Paradoxe EcoCredit:
    Situation initiale :
    - Zone rurale = pas de transport en commun
    - Vieille maison = isolation défaillante
    - Revenu moyen = pas de capital travaux
    ↓
    Score carbone :
    - Voiture = émissions élevées
    - Chauffage fioul = émissions élevées
    - Score = BAS (mauvais)
    ↓
    Demande prêt rénovation :
    - Isolation = réduction chauffage
    - Rénovation = réduction empreinte
    ↓
    Décision IA :
    - Score bas = refus crédit
    - Pas de rénovation possible
    - Reste situation émettrice
    ↓
    [PARADOXE ÉSG]
    ↓
    [BLOCAGE TRANSITION ÉCOLOGIQUE]
```

### 2.2 Incident 2 — Inégalité Riches/Pauvres

| Élément | Détail |
|:---|:---|
| **Source** | ONG |
| **Accusation** | **"Crédit social environnemental"** |
| **Mécanisme** | Riches achètent crédits carbone ailleurs pour compenser |
| **Impact** | **Pénalisation classes populaires** |
| **Inégalité** | Accès crédit inégal face au droit bancaire |

#### Analyse — Discrimination ÉSG

```
Inégalité EcoCredit:
    Riche :
    - Score carbone élevé (jet privé, yacht)
    - Achète crédits carbone (compensation)
    - Score ajusté = bon
    - Obtient crédit à taux préférentiel
    ↓
    Pauvre :
    - Score carbone élevé (voiture vieille, chauffage fioul)
    - Pas de capital achat crédits carbone
    - Score = mauvais
    - Refus crédit ou taux majoré +2%
    ↓
    [DISCRIMINATION ÉSG]
    ↓
    [INÉGALITÉ ACCÈS CRÉDIT]
```
