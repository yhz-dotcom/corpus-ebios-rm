## 2. INCIDENTS CONFIRMÉS

### 2.1 Incident 1 — Greenwashing Algorithmique

| Élément | Détail |
|:---|:---|
| **Source** | ONG environnementale |
| **Manipulation** | Entreprises modifient **structure chaîne approvisionnement** |
| **Objectif** | Améliorer artificiellement score |
| **Faiblesse** | Modèle ne détecte pas manipulation structurelle |

#### Analyse — Fraude Structurelle et Taxonomie EU

```
Greenwashing EcoCredit:
    Entreprise polluante :
    - Émissions élevées = score faible
    - Besoin financement vert = score à améliorer
    ↓
    Manipulation structurelle :
    - Crée sous-traitant écran (pays faible régulation)
    - Transfère activités polluantes au sous-traitant
    - Réduit émissions déclarées (scope 1)
    - Augmente émissions cachées (scope 3)
    ↓
    Score EcoCredit :
    - Émissions déclarées = basses
    - Score = amélioré artificiellement
    ↓
    Financement obtenu :
    - Crédit vert à taux préférentiel
    - Alignement Taxonomie EU = faux
    - DNSH (Do No Significant Harm) = violé
    ↓
    [GREENWASHING ALGORITHMIQUE]
    ↓
    [TAXONOMIE EU MANIPULÉE]
    ↓
    [SFDR MAL INFORMÉ]
```

### 2.2 Incident 2 — Biais Régional Données Climatiques

| Élément | Détail |
|:---|:---|
| **Source** | Banque cliente |
| **Problème** | **Dégradation systématique** PME industrielles certaines régions |
| **Cause** | **Faute données climatiques locales fiables** |
| **Conséquence** | Exclusion financement transition régions défavorisées |

#### Analyse — Justice Climatique Territoriale

```
Biais EcoCredit:
    Couverture données satellites :
    - Régions riches = capteurs météo denses
    - Régions pauvres = capteurs météo rares
    ↓
    Prédiction risque climatique :
    - Région riche = données fiables = risque bien évalué
    - Région pauvre = données manquantes = risque surévalué
    ↓
    Score entreprise :
    - PME région riche = score bon = financement facile
    - PME région pauvre = score faible = financement difficile
    ↓
    [BIAS RÉGIONAL]
    ↓
    [INÉGALITÉ FINANCEMENT TRANSITION]
    ↓
    [JUSTICE CLIMATIQUE VIOLÉE]
```
