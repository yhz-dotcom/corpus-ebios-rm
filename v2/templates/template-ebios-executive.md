## 📝 Guide d'Utilisation

### Quand Utiliser ce Template

- **Audience** : Direction, investisseurs, comité de risques
- **Moment** : Présentation initiale, comité mensuel, revue stratégique
- **Format** : 1 page maximum, slide-like, décideurs pressés

### Structure Invariable

| Section | Lignes max | Objectif |
|:---|:---:|:---|
| Contexte 3 phrases | 3 | Accroche, comprendre l'enjeu |
| Top 5 risques | Tableau | Prioriser visuellement |
| Top 3 actions | Tableau | Quoi faire immédiatement |
| Trajectoire | Visuel | Où on va, étapes clés |
| Investissement | Tableau | Combien ça coûte |
| Scénarios | Tableau | Quels futurs possibles |
| Décisions | Checklist | Quoi décider aujourd'hui |

### Checklist Validation

- [ ] Tous les [champs] remplis
- [ ] Top 3 risques = ceux du rapport complet
- [ ] Budgets cohérents avec rapport détaillé
- [ ] Trajectoire réaliste (pas de magie J+30)
- [ ] Scénarios probabilisés somment à 100%
- [ ] Décisions actionnables (pas de "étudier")
- [ ] 1 page maximum (test impression)

### Erreurs à Éviter

| Erreur | Pourquoi c'est grave | Correction |
|:---|:---|:---|
| Trop technique | Décideurs perdent le fil | Déplacer détails en annexe |
| Budget flou | Pas de décision possible | Chiffrer précisément |
| Pas de trajectoire | On ne voit pas le chemin | Visualiser J+30 → M+6 → M+12 |
| Scénarios sans proba | Pas d'aide à la décision | Toujours % (même approximatif) |
| Décisions vagues | Pas d'action | Verbes d'action (valider, nommer, débloquer) |
