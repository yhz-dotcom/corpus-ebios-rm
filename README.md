# Pense-bête de l'Auditeur EBIOS RM

## Guide complet pour accompagner un client dans l'analyse des risques cyber

---

## 📋 Vue d'ensemble

**EBIOS RM** (Risk Manager) est la méthodologie d'analyse de risques cyber de l'ANSSI.

### Les 5 ateliers

| Atelier | Nom | Durée | Objectif |
|---------|-----|-------|----------|
| 1 | Cadrage et Socle de Sécurité | 4-6h | Définir périmètre, biens essentiels, mesures existantes |
| 2 | Sources de Risque | 4-6h | Identifier les menaces (attaquants, cyberattaques, non-malveillant) |
| 3 | Scénarios de Risque | 6-8h | Construire et évaluer les scénarios de risque |
| 4 | Traitement du Risque | 4-6h | Définir les mesures de sécurité et risque résiduel |
| 5 | Feuille de Route | 3-4h | Construire le plan d'action et finaliser le dossier |

### Durée totale estimée
- **5 ateliers** : 3-4 jours répartis sur 3-6 semaines
- **Préparation** : 1-2 jours
- **Rédaction** : 2-3 jours
- **Total mission** : 1 à 2 mois

## 🎯 Objectifs du pense-bête

1. **Guider l'auditeur** étape par étape dans chaque atelier
2. **Faciliter la préparation** avec checklists et documents requis
3. **Fournir des templates** prêts à l'emploi
4. **Donner des astuces** de facilitation et de gestion de groupe
5. **Assurer la qualité** des livrables EBIOS RM

---

## ⚡ Quick Start

### Avant de commencer
1. Lire la **Partie 1 - Préparation**
2. Planifier les 5 ateliers avec le client
3. Rassembler les documents du client
4. Préparer les outils et templates

### Pendant les ateliers
1. Suivre le **guide de l'atelier** correspondant
2. Utiliser les **templates** fournis
3. Prendre des notes dans le **compte-rendu**
4. Valider avec le client à chaque étape

### Après les ateliers
1. Rédiger le **dossier EBIOS RM**
2. Préparer la **présentation direction**
3. Établir la **feuille de route**
4. Planifier le **suivi annuel**
